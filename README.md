# Proyecto-de-QA-Testing-a-pagina-Pedidos-YA-Uruguay
Proyecto de testing manual sobre la plataforma PedidosYa, con ejecución de 12 casos de prueba funcionales (positivos y negativos). Se identificaron 8 bugs relacionados con carrito, navegación, UI e inputs, validando flujos críticos de usuario y comportamiento general de la aplicación en un entorno de e-commerce real.
